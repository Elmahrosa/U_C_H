Pilot Deployment (3–6 months)

The pilot is a paid validation phase—limited scope, institution-isolated deployment, designed to convert into full licensing or ownership transfer.

Pilot scope (included)
  • Teleconsultation engine
  • WebRTC video consultations
  • Role-based authentication
  • Secure patient records storage
  • Admin dashboard
Pilot success metrics
  • ✓ ≥ 95% uptime
  • ✓ Verified clinician usage
  • ✓ No critical security incidents
  • ✓ Documented operational improvement
  • ✓ Executive intent to expand
  • ✓ Institution retains full data ownership during pilot

Excluded from pilot (roadmap modules): blockchain notarization, IoT vitals, insurance API integration, AI triage assistant.
